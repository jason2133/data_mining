# data_mining
Korea Univ. / Data Mining / IMEN321 / 2019 Spring

* Data Mining Term Project : **Deep Learning을 활용한 10종류의 이미지 분류**


* Jupyter Notebook 위에서 PyTorch를 활용하여 구현하였습니다.
* Jupyter Notebook 폴더 안의 .ipynb 형식의 파일에 제 코드들이 있습니다.
* Datasets 폴더의 경우 제가 이 Deep Learning Model을 구현하는데 있어 활용한 데이터셋들입니다. 폴더 안에는 Train 폴더와 Test 폴더가 있습니다. Train 폴더는 말 그대로 Train을 위해 활용된 데이터이고, Test 폴더는 Test를 위해 활용된 데이터입니다.
* Deep Learning Model을 Train 시킨 이후 Test 결과 99%의 정확도가 도출되었습니다.
* 자세한 것은 수업 시간에 제출한(할) 보고서에 서술하였습니다.
